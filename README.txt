VEGETABLE SHOP PROJECT (MERN)
=============================

SETUP INSTRUCTIONS:

1. PREREQUISITES:
   - Install Node.js
   - You need a MongoDB collection named 'products' with some items.
     Example Document: { "name": "Carrot", "price": 40, "unit": "kg", "image": "URL" }

2. BACKEND:
   - Open terminal in /backend folder
   - Run: npm install
   - Run: node server.js (Starts on port 5000)

3. FRONTEND:
   - Open terminal in /frontend folder
   - Run: npm install
   - Run: npm start (Starts on port 3000)

The MongoDB URL provided is already integrated in backend/server.js.
